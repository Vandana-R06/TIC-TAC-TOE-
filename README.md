# 🎮 Tic Tac Toe Game

An interactive and responsive Tic Tac Toe web application built using **HTML, CSS, and JavaScript** with modern UI and enhanced gameplay features.

---

## 🚀 Features

* 🎮 Two-player mode
* 🤖 Play vs Computer (basic AI)
* 🏆 Win detection with highlight
* 🤝 Draw detection
* 📊 Scoreboard (tracks Player X & O wins)
* 🔄 Restart game functionality
* 🔁 Reset score option
* 🎨 Modern UI with hover effects
* 📱 Fully responsive design
